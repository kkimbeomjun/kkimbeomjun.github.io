주민등록등본을 주민센터에 직접 가지 않고 인터넷으로 무료 발급받는 방법을 PC와 모바일로 나누어 설명합니다.

## 주민등록등본이란?

주민등록등본은 세대주 및 세대원의 성명, 주소, 생년월일 등이 기재된 공식 문서로, 취업, 대출, 전입신고 등 다양한 행정 업무에 필요합니다.

## 인터넷 발급의 장점

- **무료**: 수수료 없음 (주민센터 방문 시 400원)
- **24시간 발급 가능**: 언제든지 출력 가능
- **방문 불필요**: 집에서 바로 해결

## PC로 발급받는 방법 (정부24)

### 준비물
- 공동인증서(구 공인인증서) 또는 간편인증 (카카오, 네이버, PASS 등)
- 프린터 (출력 필요 시)

### 절차

1. **정부24 접속** — www.gov.kr 접속
2. **로그인** — 우측 상단 로그인 클릭
   - 간편인증: 카카오, 네이버, PASS, 삼성패스 등 선택
   - 공동인증서: 기존 방식으로 로그인
3. **검색창에 "주민등록등본" 입력** — 검색 후 신청 클릭
4. **발급 정보 입력**
   - 주민등록번호 뒷자리 공개 여부 선택
   - 세대원 표시 여부 선택
   - 용도 입력 (선택사항)
5. **수령 방법 선택** — "온라인 발급(본인 출력)" 선택
6. **PDF 저장 또는 출력**

> **주의**: 발급된 등본에는 위변조 방지 마크가 포함되어 있어 공식 문서로 인정됩니다.

## 모바일로 발급받는 방법

### 방법 1: 정부24 앱

1. **정부24 앱** 설치 (App Store / Google Play)
2. 앱 실행 후 로그인
3. 하단 메뉴 > 서비스 신청
4. "주민등록등본" 검색 및 신청
5. PDF 저장 후 이메일 또는 프린터로 출력

### 방법 2: 카카오톡으로 발급

1. 카카오톡 앱 실행
2. 하단 더보기(…) 클릭
3. 증명서 발급 서비스 선택
4. 주민등록등본 선택 후 인증
5. PDF 발급

### 방법 3: PASS 앱

1. PASS 앱 실행
2. 증명서 발급 메뉴 선택
3. 주민등록등본 발급 후 저장

## 발급 시 자주 하는 실수

### 세대원 전체 표시 확인
다른 가구원 정보가 필요한 경우 "세대원 전체" 옵션을 선택해야 합니다.

### 주민번호 뒷자리 표시 여부
제출처에 따라 뒷자리 표시 여부가 다르므로 미리 확인 후 발급하세요.

### 발급일 확인
대부분의 기관에서 발급일로부터 3개월 이내의 서류를 요구합니다.

## 주민센터 방문이 필요한 경우

- 타인의 등본을 발급받을 때 (대리인 서류 필요)
- 인터넷 환경이 없거나 인증 수단이 없을 때
- 영문 주민등록등본이 필요할 때

## 마무리

인터넷 발급은 무료인 데다가 24시간 이용 가능해서 훨씬 편리합니다. 정부24 또는 카카오톡을 이용하면 5분 이내로 발급받을 수 있으니, 다음번에는 주민센터에 가지 말고 온라인으로 해결해 보세요.
